# HTML : Quais elementos estão sendo usados ?
Div : selecionar uma parte especifica 
class : se junta com a div selecionando uma parte especifica, e endicando um nome a parte selecionada, para tudo que for usado ficara igual 
Id : denomina algo criando caracteristicas proprias 
nav : criar um menu
h1 :  criar um titulo, deixando es palavras maiores 
p : criar um paragrafo 
li : criar topicos no meu menu

# CSS : Quais propriedades estão sendo usados ?
body : da uma pesonalidade ao corpo so site 
text-align : centraliza meu titulo
margin : uma linha determinada para não passar de um pondo especifico
width : largura 
height : altura
padding : o tamanho do espasamento da borda 
color : cor do elemento 
background-image : colocar a imagem
url : o caminho da imagem
display :  exibição de um elemento
